## 1. 项目概述

一个支持移动端和电脑端的刷题工具（PWA），面向路桥交通工程中级考试备考。基于用户提供的6份交通工程专业文档，结合网络搜索的历年真题和考试大纲，生成1000道高质量模拟题。核心功能：刷题模式、错题回顾、模拟考试。

### 1.1 用户提供的文档

| # | 文件名 | 类型 |
|---|---|---|
| 1 | 《公路隧道设计规范 第二册 交通工程与附属设施》 | 行业规范 |
| 2 | 《高速公路交通工程及沿线设施设计通用规范》（JTG D80-2006） | 行业规范 |
| 3 | 《交通管理与控制》第四版 杨佩昆 | 教材 |
| 4 | 《公路交通安全》 | 教材 |
| 5 | 《公路工程技术标准》(JTG B01-2014) | 行业规范 |
| 6 | 《交通工程学》第三版 王炜 | 教材 |

### 1.2 目标用户

- 路桥交通工程中级职称考试备考人员
- 使用场景：手机端碎片化刷题 + 电脑端集中模拟考试

## 3. 数据结构

### 3.1 题目对象

```typescript
interface Question {
  id: number;                     // 1-1000 唯一编号
  type: 'single' | 'multi' | 'judge';
  category: string;               // 知识点分类
  difficulty: 1 | 2 | 3;         // 1=简单 2=中等 3=困难
  question: string;               // 题干
  options: string[];              // 选项列表
  answer: number[];               // 正确答案索引（从0开始）
  explanation: string;            // 解析
}
```

### 3.2 IndexedDB 表结构

| 表名 | 用途 | 主键 | 关键字段 |
|---|---|---|---|
| `questions` | 题库主表 | id | type, category, difficulty, question, options, answer, explanation |
| `progress` | 做题记录 | ++autoId | questionId, isCorrect, userAnswer, timestamp |
| `examResults` | 考试成绩 | ++autoId | score, total, date, answers[] |

### 3.3 题目分布

## 4. 页面与功能设计

### 4.1 导航结构

底部固定导航栏（移动端经典布局）：

```
[首页]   [刷题]   [错题]   [考试]
```

每个标签对应一个页面，使用 HashRouter 路由。

### 4.2 首页 (Home)

- **三大入口卡片：**
  - 刷题模式（大卡片 + 图标）
  - 错题回顾（大卡片 + 图标 + 当前错题数角标）
  - 模拟考试（大卡片 + 图标）
- **学习统计概览：**
  - 今日刷题数
  - 总正确率（环形图或百分比）
  - 错题总数
  - 最近考试成绩

### 4.3 刷题模式 (Practice)

**流程：**

1. 进入 → 配置刷题范围
   - 筛选：全部 / 按分类 / 按题型 / 按难度
   - 顺序：顺序刷 / 随机刷
2. 开始刷题 → 逐题展示
   - 题目卡片显示：题型标签、难度标签、题干、选项列表
   - 点击选项后：
     - 正确：选项变绿，显示 ✓，弹出解析
     - 错误：选项变红显示 ✗，同时高亮正确答案（绿色），弹出解析
   - 自动记录到 `progress` 表
3. 底部操作栏：上一题 / 下一题 / 题号进度
4. 刷完本轮 → 弹出小结（正确率、用时、错题数）

### 4.4 错题回顾 (WrongReview)

**流程：**

1. 从 `progress` 表筛选 `isCorrect=false` 的去重题目
2. 列表展示：
   - 题目缩略文字
   - 做错次数
   - 上次错误日期
   - 所属分类标签
3. 点击某题 → 进入错题刷题模式（同刷题模式，仅限错题）
4. 做对后自动从错题列表移除
5. 支持"一键清空错题本"（需二次确认）

### 4.5 模拟考试 (Exam)

**流程：**

1. **考试配置页：**
   - 题量选择：50 / 100（默认） / 150
   - 时间限制：60分钟 / 90分钟（默认） / 120分钟
   - 范围：全部 / 限定分类
2. **考试中：**
   - 顶部：倒计时 + 已答/未答计数
   - 左/右侧边栏（电脑端）或底部弹出（手机端）：题号列表
     - 已答 = 蓝色，未答 = 灰色，标记"待定" = 黄色
   - 题目卡片同刷题模式，但选完后不立即显示对错（不提示）
3. **交卷：**
   - 按钮始终可见
   - 点击交卷 → 弹出确认框，提醒未答题数
   - 确认 → 自动评分
4. **成绩单页：**
   - 总分 / 正确率
   - 正确/错误/未答统计
   - 逐题对照表（你的答案 vs 正确答案）
   - 保存到 `examResults` 表
   - "查看错题"按钮 → 跳转错题回顾

## 6. 题目生成策略

### 6.1 生成流程

1. **网络搜索：** 搜索路桥交通工程中级考试大纲、历年真题考点、常见题型
2. **PDF内容抽取：** 读取6份PDF文档，提取关键知识点、规范条文、公式、定义
3. **题目生成：** 结合真题考点分布和文档知识点，生成1000道题
4. **质量校验：** 确保每个题目选项合理、答案正确、解析清晰

### 6.2 题型要求

| 题型 | 数量 | 特点 |
|---|---|---|
| 单选题 | 850 | 4个选项，1个正确答案，考察知识点的准确记忆和理解 |
| 多选题 | 50 | 4-5个选项，2-3个正确答案，考察综合理解 |
| 判断题 | 100 | 判断陈述是否正确，附带解析说明原因 |

### 6.3 输出格式

所有题目输出为单一 `questions.json` 文件，符合上述 Question 接口定义。

## 7. PWA 配置

- Service Worker：离线缓存（题目数据 + 静态资源，用户首次加载后完全离线可用）
- manifest.json：应用名"路桥交通工程刷题"，可安装到桌面/主屏幕
- 图标：提供 192x192 和 512x512 尺寸

## 8. 非功能需求

### 8.1 性能
- 首屏加载 < 3秒（含1000题数据注入）
- 题目切换响应 < 100ms
- IndexedDB 批量写入 seed 数据时使用事务

### 8.2 兼容性
- 移动端：iOS Safari 14+, Chrome Android 90+
- 桌面端：Chrome 90+, Edge 90+, Firefox 90+

### 8.3 离线
- PWA Service Worker 缓存全部静态资源
- 题目数据在首次加载后全部存储在 IndexedDB，后续离线可用
- 做题记录离线存储，无需网络
